# Excel-Project
# 📊 US Candy Sales & Product Performance Dashboard (Excel)

## 📌 Project Overview

This project showcases interactive Excel dashboards developed to analyze **US Candy Sales Performance** and **Product Performance**. The dashboards provide valuable business insights into sales, profit, order count, product performance, regional trends, and customer purchasing behavior.

The project demonstrates advanced Excel skills including Pivot Tables, Pivot Charts, Slicers, Data Visualization, Dashboard Design, and Business Intelligence Reporting.

## Dashboard Features

### 📅 Year-wise Sales and Profit Analysis
- Compare yearly sales performance.
- Track profit growth over multiple years.
- Identify business growth trends.

### 🌎 Region-wise Sales Comparison
Comparison between USA and Canada regions:
- Atlantic
- Pacific
- Interior
- Gulf

### 🚚 Sales Distribution by Shipping Mode
Analyze revenue generated through:
- First Class
- Same Day
- Second Class
- Standard Class

### 🍫 Sales by Product Category
Category-wise sales analysis:
- Chocolate
- Sugar
- Other Products

---

## Interactive Filters

### Month Filter
- January to December

### Region Filter
- Atlantic
- Gulf
- Interior
- Pacific

These slicers allow dynamic dashboard exploration.

---

## Key Insights

✅ Pacific region generated the highest sales.

✅ Standard Class shipping contributed the most revenue.

✅ Chocolate category dominated total sales.

✅ Sales and profit showed consistent year-over-year growth.
